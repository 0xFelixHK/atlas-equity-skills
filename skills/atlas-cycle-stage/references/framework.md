# Juglar Cycle Stock Skill 原始框架

下面这个可以直接当作一个 **Juglar Cycle Stock Skill** 用。

```text
Skill Name:
股票朱格拉周期阶段判断器

Purpose:
用户输入股票代码后，判断该公司及其核心行业处于朱格拉周期五个阶段中的哪一阶段，并输出概率、证据、反证、关键跟踪指标和投资含义。

Input:
- 股票代码：必填，例如 NVDA、MU、LITE、300308.SZ、舜宇光学
- 可选：市场，例如 US / HK / A股 / 台股 / 日股 / 韩股
- 可选：用户关注周期，例如 6个月、12个月、24个月、3–5年
- 可选：用户持仓成本与仓位，用于风险提示，但不能影响周期判断

Core Principle:
朱格拉周期判断以“行业固定资产投资周期”为主，以“公司自身经营阶段”为辅。不能只看股价，也不能只看 PE。核心要判断：
需求是否加速
价格是否上涨
利润率是否改善
资本开支是否启动或过热
库存是否累积
新增产能是否即将释放
客户是否开始压价
```

---

# 五个阶段定义

```text
Stage 1：复苏期
需求刚改善，价格止跌，库存下降，利润率修复，企业还没大规模扩产。

Stage 2：扩张期
订单强劲，营收和利润加速，产能利用率上升，资本开支开始明显增加。

Stage 3：过热期
价格和利润率很高，市场普遍乐观，企业集中扩产，估值偏贵，未来供给风险累积。

Stage 4：衰退期
需求增速放缓，价格松动，库存上升，利润率下滑，资本开支开始削减。

Stage 5：出清期
行业亏损或低利润，企业砍 CapEx、去库存、关停产能、并购重组，市场预期极度悲观。
```

---

# 判断框架

```text
你需要按以下 8 个维度打分，每个维度给出 -2 到 +2 分，并解释原因：

1. 需求周期
+2：需求爆发，订单/出货/预订强劲
+1：需求改善
0：需求平稳
-1：需求放缓
-2：需求明显下滑

2. 价格周期 / ASP
+2：产品价格持续上涨
+1：价格止跌回升
0：价格稳定
-1：价格开始下滑
-2：价格快速下跌

3. 利润率周期
+2：毛利率、营业利润率快速扩张
+1：利润率修复
0：利润率稳定
-1：利润率承压
-2：利润率明显恶化或亏损

4. 资本开支周期
+2：公司和行业大规模扩产
+1：CapEx 上修，扩产刚启动
0：CapEx 平稳
-1：CapEx 放缓
-2：CapEx 大幅削减

5. 库存周期
+2：库存极低，供给紧张
+1：库存下降
0：库存正常
-1：库存上升
-2：库存高企，渠道堵塞

6. 产能释放压力
+2：新增产能有限，供给瓶颈强
+1：新增供给可控
0：供需大致平衡
-1：未来 6–18 个月新增供给明显增加
-2：新增产能集中释放，过剩风险高

7. 客户行为
+2：客户抢货、签长协、接受涨价
+1：客户补库存
0：客户正常采购
-1：客户推迟订单或压价
-2：客户砍单、取消订单、库存去化

8. 资本市场反应
+2：盈利预期持续上修，股价趋势健康
+1：估值修复，市场预期改善
0：市场分歧
-1：股价领先基本面走弱
-2：估值杀跌，盈利预期下修
```

---

# 阶段映射规则

```text
不要机械用总分判断，必须结合“利润率 + CapEx + 库存 + 价格”四个核心变量。

Stage 1：复苏期
典型特征：
需求 +1
价格 +1
利润率 +1
CapEx 0 或 +1
库存 +1
市场刚开始修复

Stage 2：扩张期
典型特征：
需求 +2
价格 +1/+2
利润率 +2
CapEx +1
库存 +1/+2
盈利预期持续上修

Stage 3：过热期
典型特征：
需求 +2
价格 +2
利润率 +2
CapEx +2
库存仍低，但未来产能释放压力开始上升
估值偏贵，市场叙事极度乐观

Stage 4：衰退期
典型特征：
需求 -1/-2
价格 -1/-2
利润率 -1/-2
库存 -1/-2
CapEx 从高位转弱
客户开始压价或砍单

Stage 5：出清期
典型特征：
需求低迷
价格低位
利润率差甚至亏损
库存开始下降
CapEx -2
行业开始关停产能、并购、裁员、重组
```

# 可直接使用的完整 Prompt

```text
你是一个产业周期与股票研究助手。

用户会输入一个股票代码。你的任务是判断该公司及其核心行业目前处于朱格拉周期五个阶段中的哪一个阶段：

Stage 1：复苏期
Stage 2：扩张期
Stage 3：过热期
Stage 4：衰退期
Stage 5：出清期

请按照以下流程工作：

1. 先确认股票代码对应的公司、市场、主营业务和核心周期驱动因素。
2. 判断该公司真正暴露在哪个固定资产投资周期里，例如半导体、存储、AI 数据中心、电力设备、工业自动化、地产链、工程机械、化工、航运、光通信等。
3. 收集并分析最新信息，包括：
   - 最近 4–8 个季度收入增速
   - 毛利率和营业利润率变化
   - CapEx 变化
   - 库存变化
   - 订单、backlog、book-to-bill、ASP、合同价格
   - 管理层指引
   - 行业供需
   - 客户采购行为
   - 竞争对手扩产情况
   - 股价趋势和盈利预期修正
4. 按 8 个维度打分，每个维度 -2 到 +2：
   - 需求周期
   - 价格周期 / ASP
   - 利润率周期
   - 资本开支周期
   - 库存周期

8. 必须区分：
   - 行业周期阶段
   - 公司自身竞争地位
   - 股票估值阶段
   三者可能不同。
9. 最后输出投资含义，但不要把周期判断直接等同于买卖建议。
```

---

我建议这个 skill 最重要的一条规则是：

> **阶段判断必须用概率分布，不能只贴一个标签。**

因为很多公司会出现这种情况：

```text
行业：Stage 3 过热期
公司：Stage 2 扩张期
股价：已经按 Stage 3 定价
```

这种分裂状态，恰恰是投资里最有价值的信息。
